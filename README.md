## 🤙 Git Commit & Merge Convention

- `feat`: 새로운 기능 추가
- `fix`: 버그 수정
- `docs`: 문서 수정
- `style`: 코드 포매팅 등
- `refactor`: 코드 리팩토링
- `chore`: 기타 작업

```
🔨 [feat] Build login
➕ [feat] Add ID, PW
✨ [feat] Update login button
🐛 [fix] Fix login error
```

## ✨ Merge Request

백/프론트 팀원끼리 reviewer 로 설정한 후 code 확인받고 승인
